# Student Placement Prediction using Machine Learning

## Project Overview
This project predicts whether a student will get placed based on academic and skill-based features.

## Features Used
- CGPA
- Internships
- Projects
- Communication Skills
- Technical Skills
- Aptitude Score
- Backlogs

## Machine Learning Model
- Random Forest Classifier
- Accuracy: ~80-90%

## How to Run

1. Install requirements:
pip install -r requirements.txt

2. Train model:
python train_model.py

3. Predict:
python predict.py
